Technologies: AngularJS
During summer of 2015 I had some free time and decided I was going to rebuild my website. My old site was thrown together in about four days from bootstrap templates and I built it when I had only a very basic working knowledge of HTML/CSS and no JavaScript experience. I remember going back to it after a semester and not even wanting to work on it/make it better because the code was so bad. Thus, I decided to create this site!
Page Animations
I didn't want a single page scroller like my old website. At the same time, I wanted to learn how to animate page transitions, so I decided to go with pages that would slide up and slide down when opened/closed.
Content
Was Angular necessary? No, I'm aware that for a personal web page like mine it wasn't, but this was more of a learning experience for me. I made this website (1) for myself because I wanted to familiarize myself with the MVC model and get better at web design (2) for others to look at projects I've done and what I can do. I also wanted something that people would look at and say, "this feels very you." I'm proud to present the finished product, so I hope you enjoy it!
I really love mouseover effects if that wasn't apparent already.
Takeaways
  • Page animations make everything look more interesting
  • Drawing arrows and x's is perplexing
  • Building things from scratch is really rewarding
  • Never going back to bootstrap
